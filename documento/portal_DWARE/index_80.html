publicar_noticiause caseID Caso de Uso: 003

Nombre Caso de Uso:publicar noticia.

Fecha creación: Noviembre 14 de 2008
creado por :

Fecha modificación:
Modificado por :

Objetivo: Ingresar una nueva noticia al portal.

Actores:
Administrador del sistema.

Precondiciones:
1. El usuario debe haber iniciado una session.
2. El sistema debe estar en la pagina principal del modulo de administración.

Poscondiciones:
La noticia que se ha publicado se podra ver en la pagina principal del portal.

Flujo Normal:
1.El usuario selecciona en el menu, administrar noticias.
2.El sistema va a la pagina principal del modulo administrar noticias.
3.El usuario selecciona en el menu, la opcion ingresar noticia.
4.El sistema muestra un formulario, para ingresar los datos de la noticia.
5.El usuario selecciona el tipo de noticia, ingresa el titulo, el cuerpo de la noticia y presiona el boton aceptar.
6.El sistema guarda el registro en la base de datos y muestra la pagina principal del modulo administrar noticia, mostrando una lista con la noticia que se acaba de ingresar

Flujos Alternativos:

Resultado:
Se guarda un registro de noticia que se puede visualizar en la pagina principal del portal dependeindo del tipo de noticia.
publicar_noticiaoperationPermita a un usuario registrado y con privilegios, ingresar y Publicar nuevas noticias ingresando el tipo, el titulo y el contenido de las noticias.
las cuales se veran publicadas en la pagina principal del portal.
