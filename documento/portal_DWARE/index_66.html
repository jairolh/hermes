borrar_activouse caseID Caso de Uso: 010

Nombre Caso de Uso: Borrar Activo.

Fecha creación: Noviembre 24 de 2008
creado por : Jairo Lavado Hernámdez.

Fecha modificación:
Modificado por :

Objetivo:
Eliminar el registro de activos del sistema.

Actores:
Gestor RIA.

Precondiciones:
1. El usuario debe haber iniciado una session.
2. El sistema debe estar en la pagina principal del modulo de Activos, con un listado de activos.

Poscondiciones:
Se modifico los datos del registro de un activo.

Flujo Normal:
1.El usuario ubica el activo en la lista y selecciona en la lista frente al activo, la opcion borrar Activo.
2.El sistema muestra un mensaje de confirmación de borrado.
3.El usuario Acepta el borrado del activo.
4.El sistema borra el registro en la base de datos y muestra la pagina principal del modulo Activos, mostrando una lista general de activos.

Flujos Alternativos:

* El usuario seleccioan cancela en el mensaje de confirmación.

4.El sistema muestra la pagina principal del modulo Activos, mostrando una lista general de activos, sin borrar registros.

Resultado:
Se Borra el registro de un activo de la base de datos del sistema.
